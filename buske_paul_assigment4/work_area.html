Work Area

This lesson covered the basics of Photoshop. The intent was to create a birthday card. I adjusted the brightness and contrast using the "Adjustments" tab. Then I used the "Rectangle Marquee tool" to create a ribbon. I filled the ribbon using a brush and eyedropper tools to sample the color from the Happy Birthday ribbon. Then I added my name on the ribbon using the "Horizontal Typography" tool.
